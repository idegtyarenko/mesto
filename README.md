# Проект: Mesto

## О проекте
Учебный проект в рамках [Яндекс Практикума](https://practicum.yandex.ru/web/). Представляет собой адаптивный сайт — каталог интересных мест.

## Использованные технологии
* Flexbox
* Grid
* Media-queries
* Манипулирование классами с помощью JS
